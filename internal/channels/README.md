# Channels

## Overview
This package is a collection of all the channels that are used between packages, and some inside packages. They are all on the format:

`var` `<`channelName`>`F`<`sourceModule`>`T`<`destinationModule`>`\
F = from, T = to\
Modules:
* nm  - Network Manager
* om  - Order Manager
* fsm - FSM
* hm  - Hardware Manager
